#Selection Sort
Sorting algorithm is used to rearrange a given array or list of elements in an order
It works by selecting the smallest element or assuming the any number as the smallest element and replacing it with the (swapping) smaller element, until the list ends.

Steps to perform the selection sort:
1. Define the value of the first element as a start and swap it with the smallest element i.e swap first position with the smallest element in the first iteration.
2. then, find the second smallest by iterating over the remaining elements and swap it with the second element.
3. Keep repeating the above process until you finish the iteration and we get all elements moved to the correct position.

Let's perform the sorting algorithm on the following list:
given list = L = [74, 32, 89, 55, 21, 64]
sorted list = L = [21, 32, 55, 64, 74, 89]
